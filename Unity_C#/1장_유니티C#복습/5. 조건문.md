조건문 
------

## if문

**한가지 or 두가지 경우로 분기**
```예시
if(PlayerHealth == 100)
{
  Debug.Log ("Player has full health");
}
```
>(재생버튼을 눌러서 실행가능)<br>
>Debug.Log 는 콘솔에 문자열 출력<br>
>조건이 true일 때만 실행<br>

if-else문
```예시
if(PlayerHealth == 100)
{
  // true 일때 실행
}
else
{
  //false 일때 실행
}
```

연산자 
* ==
*  >,<
* !=
* &&(and), ||(or)
